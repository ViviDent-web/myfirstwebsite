# My First Website

Il mio primo sito web realizzato durante un corso di **HTML**, **CSS** e introduzione a **JavaScript**.

## 📖 Descrizione

Questo progetto rappresenta il mio primo approccio allo sviluppo web. L'obiettivo era imparare le basi della creazione di una pagina web, partendo dalla struttura HTML, passando per lo stile con CSS e aggiungendo alcune funzionalità con JavaScript.

Durante lo sviluppo ho messo in pratica i concetti fondamentali appresi nel corso, come:

* Struttura di una pagina HTML
* Utilizzo di elementi semantici
* Layout e styling con CSS
* Gestione di colori, font e spaziature
* Introduzione a JavaScript per semplici interazioni

## 🛠️ Tecnologie utilizzate

* HTML5
* CSS3
* JavaScript (base)

## 🎯 Obiettivi del progetto

* Applicare le basi dello sviluppo web.
* Comprendere la relazione tra HTML, CSS e JavaScript.
* Realizzare un sito web statico completamente funzionante.
* Acquisire familiarità con Git e GitHub per la gestione del codice.

## 🚀 Come eseguire il progetto

Puoi utilizzare uno dei seguenti metodi.

Metodo 1 - Clonare il repository con Git

1. Clona il repository:

```bash
git clone https://github.com/ViviDent-web/myfirstwebsite.git
```

2. Apri la cartella del progetto.

3. Apri il file `index.html` con il tuo browser preferito.

Non sono necessarie installazioni o dipendenze aggiuntive.

Metodo 2 - Scaricare il progetto come ZIP
Apri il repository su GitHub.
Clicca sul pulsante Code.
Seleziona Download ZIP.
Estrai il contenuto del file ZIP in una cartella.
Apri il file index.html con il tuo browser preferito.

Non sono necessarie installazioni, dipendenze o configurazioni aggiuntive.
